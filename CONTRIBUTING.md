# Contributing to Spectrum (Commercial)

This repository is a commercial-grade deterministic computation engine.

## Contribution Policy
- Contributions require explicit written authorization
- All contributions are subject to IP assignment to AttraQter Labs LLC
- No anonymous or drive-by contributions accepted

## Prohibited
- Introducing nondeterminism
- Floating point arithmetic
- Probabilistic logic
- Hidden state or side effects
- External dependencies without approval

All submissions are audited for determinism, replayability, and license compliance.
