# Changelog

All notable changes to Spectrum are documented in this file.

The format is based on Keep a Changelog,
and this project adheres to deterministic versioning.

---

## [v3.0-public-boundary] - 2026-01-XX

### Established
- Explicit public system boundaries
- Deterministic, non-learning core frozen
- Interface and contract layer stabilized

### Guarantees
- Bit-identical reproducibility
- No randomness, time, environment, or floating-point dependence
- Exact arithmetic only (integers / rationals)
- Immutable state transitions
- Canonical ordering of all collections

### Clarifications
- Spectrum is **not** machine learning
- Spectrum is **not** probabilistic
- Spectrum is **not** adaptive
- Spectrum is an audit-grade deterministic engine

This release is suitable for:
- Regulated environments
- Safety-critical systems
- Scientific verification
- Compliance and audit pipelines
