# Spectrum Boundaries

Spectrum is a deterministic measurement and audit engine.

## What Spectrum IS
- Deterministic
- Non-learning
- Replayable
- Exact (integer / rational arithmetic only)
- Auditable and traceable
- Suitable for regulated and safety-critical domains

## What Spectrum IS NOT
- A machine learning system
- A probabilistic engine
- An optimizer
- A predictor
- An adaptive or heuristic system

Spectrum does not infer, guess, or approximate.

All behavior is explicit, declared, and reproducible.
